# 🎬 SQL Movie Analysis Project

## 📌 Project Overview

This project demonstrates how SQL can be used to analyze a movie database and solve real-world business problems using PostgreSQL. It includes database design, data import, and analytical SQL queries to generate valuable insights from movie, user, rating, genre, and actor data.

The project showcases fundamental and advanced SQL concepts including joins, aggregate functions, GROUP BY, CASE statements, window functions, ranking, filtering, and sorting. It is designed as a portfolio project to demonstrate practical SQL and data analysis skills.

## 📊 Business Questions Answered

### ⭐ Top 5 Highest Rated Movies
Retrieve the five highest-rated movies based on user ratings.

### ⭐ Movies Rated Above 9 by Indian Users
Identify movies highly rated by users from India.

### ⭐ Number of Movies per Genre
Analyze the distribution of movies across genres.

### ⭐ Actor with the Highest Number of Movies
Find the actor who has appeared in the greatest number of movies.

### ⭐ Average Rating Given by Each User
Evaluate user rating behavior by calculating each user's average rating.

### ⭐ Movie Release Ranking
Rank movies according to their release year using SQL Window Functions.

### ⭐ Movie Classification
Classify movies as **Classic**, **Modern**, or **New** using CASE statements.

### ⭐ Recommended Movie of the Day
Recommend the highest-rated movie from the database.
